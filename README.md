# CSE412 Movies Project

## Requirements

- Flask 3.1.2 (pip install Flask)

## Instructions

1. Clone repository into a local directory
2. Start app: ```python run.py```
3. Access the app at: ```https://localhost:5000```
4. Stop the app via CTRL+C
